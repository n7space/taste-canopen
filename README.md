# CANopen TASTE component and demonstration project

Re-usable TASTE component providing CANopen node functionality, developed as a part of "Model-Based Execution Platform for Space Applications" project (contract 4000146882/24/NL/KK) financed by the European Space Agency.

It wraps [CANopen](https://n7space.com/canopen/) library, originally developed by Lely Industries and adapted for space applications by N7 Space, in a TASTE component.  The component itself is located in the **component/CANopen** directory. A demonstration project is provided in **test-project/CANopen-demo** directory. For executing the basic tests, please refer to the demonstration project's README and note the pre-requisites, including the necessary kernel features and TASTE add-ons.

For the component's README, [go here](./component/CANopen/README.md).

For the demonstration project's README, [go here](./test-project/CANopen-demo/README.md).
